# Cognifyz Software Development Internship

This repository contains a collection of Java-based practice projects completed during the internship. Each project focuses on a different programming concept and is kept simple, structured, and easy to run.

## Included Projects

1. Task 1 - Guess the Number
   - A console-based number guessing game.
   - Demonstrates random number generation and user input handling.

2. Task 3 - CRUD Task Manager
   - A console application for creating, viewing, updating, and deleting tasks.
   - Shows a simple service-based Java structure.

3. Task 5 - CRUD with File Storage
   - Extends the task manager with persistent storage using a text file.
   - Demonstrates file I/O and data persistence.

4. Task 6 - Web Scraper
   - A basic web scraper that reads data from a sample website and saves the results.
   - Uses Java's standard library without external packages.

## Repository Structure

- Task-1-Guess-the-Number/ - Number guessing game
- Task-3-CRUD-Task-Manager/ - In-memory task manager
- Task-5-CRUD-with-File/ - Task manager with file-based persistence
- Task-6-Web-Scraper/ - Simple web scraper

## How to Run

Each task is independent. Open the relevant folder and follow the instructions in its README file.

## Requirements

- Java 8 or newer
- A terminal or command prompt

No external dependencies are required for these projects.
