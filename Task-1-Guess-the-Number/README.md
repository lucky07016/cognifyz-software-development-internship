# Task 1 - Guess the Number

This project is a simple console-based game written in Java. The program generates a random number between 1 and 100, and the user tries to guess it.

## Features

- Random number generation
- Input handling through the console
- Feedback for high, low, and correct guesses
- Option to play again after each round

## Project Structure

- src/Main.java - Entry point of the application
- src/Game.java - Main game logic
- src/RandomNumberGenerator.java - Random number generation helper

## How to Run

From the project root:

```bash
cd Task-1-Guess-the-Number
javac -d out src/*.java
java -cp out Main
```

## Notes

The game is intentionally simple and is meant to demonstrate basic Java programming concepts.
